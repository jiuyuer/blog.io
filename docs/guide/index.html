# 介绍

TinyWeb 是基于目前最受欢迎、技术最前沿的 Vue2.0，配合使用 VUX(H5)、ElementUI(PC) 组件库进行封装的简单、灵活、快速、多功能、高效、轻量级的前端集成解决方案，其主要服务于 H5 的开发以及企业级中后台产品。它首先让业务开发变得简单，给程序猿带来更快、更炫、更灵活、更轻松的开发体验。它让系统页面加载速度更快。

# 特性

# 简单

  • 前后端分离：只需专注前端开发。
  • 组件使用方便：符合人性化 UI 设计。
  • 关注点分离：通过声明式的编程方式，让开发变得更加简单。
  • 数据驱动：MVVM 模式提供了双向绑定，无需操作 dom（可以有效避免内容泄露的发生），使开发变得简单。

# 灵活

  • 插件式开发：高内聚解耦，灵活易扩展。
  • 个性化定制：主界面、标题栏、皮肤样式定制。
  • 多语言支持：资源包管理方式满足国际化需求。

# 快速

  • 基于 VUE 框架开发：打造轻量级的组件库。
  • 快速开发部署：提供了脚手架快速创建应用。
  • MVVM 模式：页面加载速度更快。
  • 按需加载：较少 http 请求，较小文件大小，从而提高页面加载速度。

# 高效

  • 易开发，易扩展，易维护

# 多功能

  • 提炼了典型的业务模型，提供了丰富的功能组件

# 主题

  • 支持多种动态换肤功能

# 功能

- 登录/注销
- 权限验证
- 动态侧边栏
- 动态面包屑
- 快捷导航(标签页)
- 国际化多语言
- 富文本编辑器
- Svg 图标
- 图片上传
- 导入/导出 excel
